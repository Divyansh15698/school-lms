# School LMS

A Learning Management System for Classes 6–12.

## Roles

- Developer
- Teacher
- Student

## Features

- Books Upload
- Notes Upload
- Syllabus Upload
- Videos Upload
- Answer Keys
- Quiz System
- Reports
- Daily Streaks
- Points System
- Leaderboard
- AI Assistant
- AI Doubt Solver

## Technology

- React
- TypeScript
- Node.js
- PostgreSQL 17
- Supabase Storage
