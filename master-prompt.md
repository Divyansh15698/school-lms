# School LMS Master Prompt

Build a complete School LMS for Classes 6-12.

Roles:
- Developer
- Teacher
- Student

Developer:
- Create Teacher and Student accounts
- Generate IDs and passwords
- Reset passwords
- Force password changes
- View login history
- Manage all users

Teacher:
- Upload books
- Upload syllabus
- Upload notes
- Upload videos
- Upload answer keys
- Create quizzes
- View student reports

Student:
- Access learning content
- Take quizzes
- Earn points
- Daily streaks
- Leaderboard
- Unlock titles

Content Structure:
Class → Subject → Chapter → Topic

Database:
- PostgreSQL 17

Storage:
- Supabase Storage

Buckets:
- books
- syllabus
- notes
- videos
- answer-keys

Features:
- JWT Authentication
- Role Based Access Control
- AI Assistant
- AI Doubt Solver
- Reports
- Streaks
- Points System
- Leaderboard
- Responsive Design
